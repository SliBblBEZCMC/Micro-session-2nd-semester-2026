1. Фундаментальная последовательность в $\mathbb{R}^n$.
   > [!definition] Фундаментальная последовательность (в $\mathbb{R}^n$)
> Последовательность $\{x_k\}_{k=1}^\infty \subset \mathbb{R}^n$ называется **фундаментальной**, если:
> $$\forall \varepsilon > 0 \;\; \exists N \in \mathbb{N} \;\; \forall k, m \ge N: \|x_k - x_m\| < \varepsilon$$
> 
> 
> > [!note] Свойство
> > Если $\{x_k\}$ сходится, то она фундаментальна. - по сути можно это выдавать за определение

2. Определение функции Лагранжа. Определение стационарной точки функции Лагранжа.
>[!info] Функция Лагранжа
>Пусть $u: \mathbb{R}^n \to \mathbb{R}$, $f_1, \dots, f_m: \mathbb{R}^n \to \mathbb{R}$, $m < n$.
>$$\begin{cases}
f_1(x^1, \dots, x^n) = 0 \\
\dots \\
f_m(x^1, \dots, x^n) = 0
\end{cases} \text{ — уравнения связи (независимы)}$$
>Пусть $\lambda_1, \dots, \lambda_m \in \mathbb{R}$.
Функция $L(x^1, \dots, x^n, \lambda_1, \dots, \lambda_m) = u(x^1, \dots, x^n) + \lambda_1 f_1(x^1, \dots, x^n) + \dots + \lambda_m f_m(x^1, \dots, x^n)$ называется **функцией Лагранжа**.

>[!info] Стационарная точка функции Лагранжа
>Пусть дана функция Лагранжа для задачи на условный экстремум:
$$\mathcal{L}(\mathbf{x}, \boldsymbol{\lambda}) = f(\mathbf{x}) + \sum_{k=1}^m \lambda_k \, g_k(\mathbf{x})$$
**Стационарная точка функции Лагранжа** — это точка $(\mathbf{x}^*, \boldsymbol{\lambda}^*) \in \mathbb{R}^{n+m}$, в которой все частные производные функции $\mathcal{L}$ равны нулю.

3. Верхняя и нижняя суммы Дарбу. Критерий интегрируемости по Риману.
> [!abstract] Верхняя и нижняя суммы Дарбу
> Пусть $f: \Omega \to \mathbb{R}$ интегрируема по Риману на измеримом по Жордану множестве $\Omega$. Пусть $T$ — разбиение $\Omega$, $T = \{\Omega_i\}$. Пусть $m_i = \inf_{\Omega_i} (f(x))$, $M_i = \sup_{\Omega_i} (f(x))$.
> Тогда:
> * $s_T = \sum_{i} m_i \cdot m(\Omega_i)$ — нижняя сумма Дарбу функции $f$ на множестве $\Omega$ при разбиении $T$.
> * $S_T = \sum_{i} M_i \cdot m(\Omega_i)$ — верхняя сумма Дарбу функции $f$ на множестве $\Omega$ при разбиении $T$.

> [!info] Теорема (критерий интегрируемости по Риману)
> $f: \Omega \to \mathbb{R}$ интегрируема по Риману на измеримом по Жордану множестве $\Omega \iff \forall \varepsilon > 0 \ \exists \delta > 0$ такое, что для любого разбиения $T$ с мелкостью $\lambda_T < \delta \implies |S_T - s_T| < \varepsilon$.

4. Тригонометрическая форма записи комплексного числа. Показательная форма записи комплексного числа. Формула извлечения корня из комплексного числа.
>[!info] Тригонометрическая форма записи
>$$z = |z|(\cos \varphi + i \sin \varphi)$$

>[!info] Показательная форма записи
>$$z = |z|e^{i\varphi}$$

>[!info] Формула извлечения корня из комплексного числа
>**Извлечение корня $n$-й степени** ($n \in \mathbb{N}$)
Пусть $z = r e^{i\varphi}$, $r > 0$. Тогда: $$\sqrt[n]{z} = \sqrt[n]{r} \cdot e^{i\frac{\varphi + 2\pi k}{n}}, \quad k = 0, 1, \dots, n-1$$
>
>- $\sqrt[n]{r}$ — арифметический корень из положительного числа $r = |z|$
>- $k$ даёт $n$ различных корней
>- Аргументы корней: $\frac{\varphi}{n} + \frac{2\pi k}{n}$
