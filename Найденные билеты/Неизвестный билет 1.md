1. Отрезок в $R^{n}$.
> [!definition] Отрезок в $\mathbb{R}^n$
> Отрезком с концами $a = (a^1, \dots, a^n)$, $b = (b^1, \dots, b^n)$ называется множество
> $$\{ x \in \mathbb{R}^n : x^i = a^i + b^i \cdot (1 - t), \ t \in \mathbb{R},\ 0 \le t \le 1 \}$$

2. Простейшее теорема неявной функции.

3. Нижняя и верхняя сумма Дарбу. Критерий интегрируемости по Риману.
> [!abstract] Верхняя и нижняя суммы Дарбу
> Пусть $f: \Omega \to \mathbb{R}$ интегрируема по Риману на измеримом по Жордану множестве $\Omega$. Пусть $T$ — разбиение $\Omega$, $T = \{\Omega_i\}$. Пусть $m_i = \inf_{\Omega_i} (f(x))$, $M_i = \sup_{\Omega_i} (f(x))$.
> Тогда:
> * $s_T = \sum_{i} m_i \cdot m(\Omega_i)$ — нижняя сумма Дарбу функции $f$ на множестве $\Omega$ при разбиении $T$.
> * $S_T = \sum_{i} M_i \cdot m(\Omega_i)$ — верхняя сумма Дарбу функции $f$ на множестве $\Omega$ при разбиении $T$.

> [!info] Теорема (критерий интегрируемости по Риману)
> $f: \Omega \to \mathbb{R}$ интегрируема по Риману на измеримом по Жордану множестве $\Omega \iff \forall \varepsilon > 0 \ \exists \delta > 0$ такое, что для любого разбиения $T$ с мелкостью $\lambda_T < \delta \implies |S_T - s_T| < \varepsilon$.

4. Определения: линейное однородное диф.уравнение высших порядков, линейное неоднородное диф.уравнение высших порядков, система линейно независимых функций, ФСР
> [!abstract] ЛОДУ высшего порядка с постоянными коэффициентами
> 
> **1. Определение:**
> Уравнение вида:
> $$y^{(n)} + p_1 y^{(n-1)} + p_2 y^{(n-2)} + \dots + p_n y = 0$$
> где $p_i \in \mathbb{R}$ — константы, $n$ — порядок уравнения, правая часть равна $0$.
> 
> **2. Характеристический многочлен:**
> Алгебраический многочлен, получаемый заменой производных $y^{(k)}$ на степени $\lambda^k$:
> $$\chi(\lambda) = \lambda^n + p_1 \lambda^{n-1} + p_2 \lambda^{n-2} + \dots + p_n = 0$$
> 
> **3. Общий вид решения:**
> Линейная комбинация функций фундаментальной системы решений (ФСР):
> $$y_{\text{общ}} = C_1 y_1 + C_2 y_2 + \dots + C_n y_n$$
> 
> ---
> 
> ### Сводная таблица для ФСР (по корням $\lambda$):
> 
> | Тип корня многочлена $\chi(\lambda)$ | Кратность $k$ | Соответствующие решения в ФСР ($y_i$) |
> | :--- | :--- | :--- |
> | **Действительный** $\lambda$ | $k = 1$ | $e^{\lambda x}$ |
> | **Действительный** $\lambda$ | $k > 1$ | $e^{\lambda x}, \; x e^{\lambda x}, \; x^2 e^{\lambda x}, \dots, \; x^{k-1} e^{\lambda x}$ |
> | **Комплексный** $\lambda_{1,2} = \alpha \pm i\beta$ | $k = 1$ | $e^{\alpha x} \cos(\beta x), \quad e^{\alpha x} \sin(\beta x)$ |
> | **Комплексный** $\lambda_{1,2} = \alpha \pm i\beta$ | $k > 1$ | $e^{\alpha x} \cos(\beta x), \; e^{\alpha x} \sin(\beta x), \dots, \; x^{k-1} e^{\alpha x} \cos(\beta x), \; x^{k-1} e^{\alpha x} \sin(\beta x)$ |

> [!example]
> 
> ## 1. Характеристическое уравнение и однородное решение
> 
> $$\lambda^2 - 5\lambda + 6 = 0$$
> 
> $$(\lambda - 2)(\lambda - 3) = 0$$
> 
> $$\lambda_1 = 2, \quad \lambda_2 = 3$$
> 
> $$y_0(x) = C_1 e^{2x} + C_2 e^{3x}$$
> 
> ## 2. Частное решение неоднородного уравнения
> 
> Правая часть: $f(x) = 5x e^{2x}$. Корень $\gamma = 2$ совпадает с корнем $\lambda_1 = 2$ (кратность $r = 1$).
> 
> $$y_p(x) = x \cdot (Ax + B) e^{2x} = (Ax^2 + Bx) e^{2x}$$
> 
> $$y_p'(x) = (2Ax + B) e^{2x} + 2(Ax^2 + Bx) e^{2x} = (2Ax^2 + (2A + 2B)x + B) e^{2x}$$
> 
> $$y_p''(x) = (4Ax + 2A + 2B) e^{2x} + 2(2Ax^2 + (2A + 2B)x + B) e^{2x} = (4Ax^2 + (8A + 4B)x + 2A + 4B) e^{2x}$$
> 
> Подстановка в исходное уравнение:
> 
> $$(4Ax^2 + (8A + 4B)x + 2A + 4B) - 5(2Ax^2 + (2A + 2B)x + B) + 6(Ax^2 + Bx) = 5x$$
> 
> $$(4A - 10A + 6A)x^2 + (8A + 4B - 10A - 10B + 6B)x + (2A + 4B - 5B) = 5x$$
> 
> $$-2Ax + (2A - B) = 5x$$
> 
> Система коэффициентов:
> 
> $$\begin{cases} -2A = 5 \\ 2A - B = 0 \end{cases}$$
> 
> $$\begin{cases} A = -2.5 \\ B = -5 \end{cases}$$
> 
> $$y_p(x) = (-2.5x^2 - 5x) e^{2x}$$
> 
> ## 3. Общее решение
> 
> $$y(x) = y_0(x) + y_p(x)$$
> 
> $$y(x) = C_1 e^{2x} + C_2 e^{3x} - (2.5x^2 + 5x) e^{2x}$$

> [!example]
> 
> ## 1. Характеристическое уравнение и однородное решение
> 
> $$\lambda^2 + 1 = 0$$
> 
> $$\lambda^2 = -1$$
> 
> $$\lambda_1 = i, \quad \lambda_2 = -i$$
> 
> $$y_0(x) = C_1 \cos x + C_2 \sin x$$
> 
> ## 2. Частное решение неоднородного уравнения
> 
> Правая часть: $f(x) = 4\sin x$. Число $\gamma = i$ совпадает с корнем $\lambda_1 = i$ (кратность $r = 1$).
> 
> $$y_p(x) = x \cdot (A \cos x + B \sin x)$$
> 
> $$y_p'(x) = (A \cos x + B \sin x) + x(-A \sin x + B \cos x) = (A + Bx)\cos x + (B - Ax)\sin x$$
> 
> $$y_p''(x) = B \cos x - (A + Bx)\sin x - A \sin x + (B - Ax)\cos x = (2B - Ax)\cos x + (-2A - Bx)\sin x$$
> 
> Подстановка в исходное уравнение:
> 
> $$((2B - Ax)\cos x + (-2A - Bx)\sin x) + (Ax \cos x + Bx \sin x) = 4\sin x$$
> 
> $$2B \cos x - 2A \sin x = 4\sin x$$
> 
> Система коэффициентов:
> 
> $$\begin{cases} 2B = 0 \\ -2A = 4 \end{cases}$$
> 
> $$\begin{cases} B = 0 \\ A = -2 \end{cases}$$
> 
> $$y_p(x) = -2x \cos x$$
> 
> ## 3. Общее решение
> 
> $$y(x) = y_0(x) + y_p(x)$$
> 
> $$y(x) = C_1 \cos x + C_2 \sin x - 2x \cos x$$
