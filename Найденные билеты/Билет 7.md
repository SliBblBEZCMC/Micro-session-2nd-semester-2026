1. Шар в метрическом пространстве. Внутренняя точка множества $R^{n}$. Открытое множество в $R^{n}$. 
> [!definition] Шар
> $B_r(a) = \{ x \in \mathbb{R}^n : d(x, a) < r \}$ — шар.

> [!definition] Внутренняя точка множества в $\mathbb{R}^n$
> Точка $x$ называется внутренней точкой множества $M$, если $x \in M \Rightarrow \exists r \in \mathbb{R}_+ : B_r(x) \in M$.

> [!definition] Открытое множество в $\mathbb{R}^n$
> $M$  называется открытым, если $M^\circ = M$.
> $M$ называют открытым множеством, если все точки этого множества открыты.

2. Определение стационарной точки. Теория о необходимых условиях локальное экстремума. 
>[!info] Стационарная точка функции Лагранжа
>Пусть дана функция Лагранжа для задачи на условный экстремум:
$$\mathcal{L}(\mathbf{x}, \boldsymbol{\lambda}) = f(\mathbf{x}) + \sum_{k=1}^m \lambda_k \, g_k(\mathbf{x})$$
**Стационарная точка функции Лагранжа** — это точка $(\mathbf{x}^*, \boldsymbol{\lambda}^*) \in \mathbb{R}^{n+m}$, в которой все частные производные функции $\mathcal{L}$ равны нулю.

3. Элементарная область относительно оси. Теория о сведении двойного интеграла Римма по элементарной области к *(к какой-то области, а определений 3 штуки)*.
   > [!abstract] Определение
> Пусть функции $\varphi, \psi : [a; b] \to \mathbb{R}$ непрерывны на отрезке $[a; b]$. Тогда область $G = \{ (x, y) : x \in [a; b], \varphi(x) \le y \le \psi(x) \}$ называется **элементарной относительно оси $Oy$**.

> [!info] Теорема (о сведении кратного интеграла к повторному на прямоугольной области)
> Пусть $f : \Pi \to \mathbb{R}$ интегрируема по Риману на $\Pi$, где $\Pi = \{ (x, y) : a \le x \le b, c \le y \le d \}$.
> Пусть $\forall x \in [a; b] \ \exists \int_{c}^{d} f(x, y) \, dy$.
> Тогда:
> $$\iint_{\Pi} f(x, y) \, dx dy = \int_{a}^{b} dx \int_{c}^{d} f(x, y) \, dy = \int_{c}^{d} dy \int_{a}^{b} f(x, y) \, dx$$

> [!info] Теорема о сведении двойного интеграла Римана по элементарной области к повторному определение области элементарной относительно оси y
> Пусть $f : G \to \mathbb{R}$ — интегрируема по Риману на $G$, где $G$ элементарна относительно оси $Oy$.
> Пусть $\forall x \in [a; b] \ \exists F(x) = \int_{\varphi(x)}^{\psi(x)} f(x, y) \, dy$.
> Тогда:
> $$\iint_{G} f(x, y) \, dx dy = \int_{a}^{b} dx \int_{\varphi(x)}^{\psi(x)} f(x, y) \, dy$$

> [!info] Теорема о сведении тройного интеграла Римана по элементарной области к повторному определение области элементарной относительно оси z
> Пусть $\Omega$ — элементарная относительно оси $Oz$ область. $f : \Omega \to \mathbb{R}$ интегрируема по Риману на $\Omega$.
> Тогда:
> $$\iiint_{\Omega} f(x, y, z) \, dx dy dz = \iint_{G} dx dy \int_{\varphi(x, y)}^{\psi(x, y)} f(x, y, z) \, dz$$

4. Определение комплексного числа. Определение множества комплексных чисел. Действительная часть, мнимая часть, модуль комплексного числа. Определение комплексного сопряжённого числа.
>[!info] Определение комплексного числа. Определение множества комплексных чисел
>**Комплексное число** — это упорядоченная пара действительных чисел $(a, b)$, для которой особым образом определены операции сложения и умножения.
Обычная запись:
$$z = a + bi$$
где:
>- $a = \operatorname{Re}(z)$ — **действительная часть**,
>- $b = \operatorname{Im}(z)$ — **мнимая часть**,
>- $i$ — **мнимая единица**, удовлетворяющая условию $i^2 = -1$.
**Множество комплексных чисел**:
$$\mathbb{C} = \{ a + bi \mid a, b \in \mathbb{R} \}$$

>[!info] Действительная часть Мнимая часть Модуль комплексного числа
>Пусть $z = a + bi$, где $a, b \in \mathbb{R}$, $i^2 = -1$.
**Действительная часть**:
$$\operatorname{Re}(z) = a$$
**Мнимая часть**:
$$\operatorname{Im}(z) = b$$
**Модуль комплексного числа**:
$$|z| = \sqrt{a^2 + b^2}$$

>[!info] Определение и свойства комплексно сопряженного числа
>**Комплексно сопряжённое число** (или просто сопряжённое) к $z = a + bi$ — это число:
$$\overline{z} = a - bi$$
где $a, b \in \mathbb{R}$, $i^2 = -1$.
**Геометрический смысл**: отражение точки $z$ относительно действительной оси на комплексной плоскости.
